Tips

1. Selection

All items are selectable. To perform actions at first select any item(s).

Or right click to select item and perform action from context menu.

2. Actions

Select any item(s) to perform actions (activated in left toolbar). This mode allows actions for multiple items.

Right click to perform action for single item from context menu.

3. Manage view

Click empty spave & drag it to any suitable position (or use arrow keys from keyboard).

Try zooming feature to get big picture of your data (or use + or - keys from keyboard).

4. Structure data

All nodes are automatically grouped by URL for your convenience.

Right click any window / group and click "expand"/"collapse" to show / hide it's contents.

5. Filter data

Use search to filter your data and get matching results. Only a few clicks to find the desired node!

Filtering by both website name and URL is supported. (e.g. accepts either of Google and www.google.com)

MindTabs

MindTabs helps to see the current browser state with a simple graphically structured tree. This tree is truly live - every branch & item within it can be changed in realtime mode.

MindTabs is a flexible tool able to solve different tasks. This tasks depend on what kind of user you are. Someone will like MindTabs for it's open/close feature. Another will reveal possibilities in suspending tabs + immediate reducing memory used by Chrome.

But for every user MindTabs is a big picture with endless deeping & grouping. So find your personal use case & give us feedback.

Learn more about key features:

Actions
Action Description
Open double click node to open tab or right click node & select "Open".
! Select active tabs to perform this action.
Close click to close tab / tab group / window or right click tab/tab group / window & select "Close".
! Select active tabs to perform this action.
Ctrl + Shift + T to open last closed tab.
Suspend click to save tab / tab group / window or right click tab/tab group / window & select "Suspend".
! Select active tabs to perform this action.
Restore click to restore suspended tab / tab group / window or right click tab/tab group / window & select "Suspend".
! Select suspended tabs to perform this action.
Delete click to delete suspended tab / tab group / window or right click tab/tab group / window & select "Suspend".
! Select suspended tabs to perform this action.

Searching

Search allows to filter all tabs and access any tab faster!


Search by title or URL (even few letters) to find tabs. Start typing any keyword from tab/tab group and get matching results.

Hover any item to know the path for every tab/tab group.


! Don't forget to clear search box to show all tabs!

Tabs, groups, windows

Multiple browser windows are supported.
From now working in few windows has become more suitable. Switch between tabs in windows, save windows & even restore them in one click.


All tabs are clustered by browser window & by URL group.
E.g. few opened tabs from Google are clustered to single group named "Google".


Area positioning: zooming & dragging.
Use zoom buttons for zooming in/out tabs area. Click empty area and drag it to any suitable position.

Hotkeys

Use hotkeys for primary access to actions like mode toggling, zooming, moving & refresh.
